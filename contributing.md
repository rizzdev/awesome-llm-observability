# Contributing

Pull requests are welcome. To add an entry:

- It has to be real, current, and genuinely useful. No dead or renamed projects, no weaker clones of something already listed.
- Keep the format: `- **[Name](https://url)** \u00b7 One line on what it does`, no trailing period.
- Add 🌿 if the project is newer and still under 500 stars.
- One entry per line, in the section it belongs to.

Open a PR and I'll take a look.
